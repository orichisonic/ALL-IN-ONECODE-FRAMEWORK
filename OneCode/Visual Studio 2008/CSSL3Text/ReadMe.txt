========================================================================
    SILVERLIGHT APPLICATION : CSSL3Text Project Overview
========================================================================

/////////////////////////////////////////////////////////////////////////////
Use:

This example demonstrates how to work with text in Silverlight using C#.
It includes the following features:

	Create a simple TextBlock.
	Create a TextBlock with multiple Runs and LineBreaks.
	Creates a TextBlock with advanced format.
	Select all text in a TextBox using code.


/////////////////////////////////////////////////////////////////////////////
Project Relation:

CSSL3Text - VBSL3Text - XAMLSL3Text

CSSL3Text and VBSL3Text expose the same UI and the same set of properties,
methods, and events, but they are implemented in different languages. 
XAMLSL3Text is different in that it shows you how to work with text in XAML
rather than code behind.
