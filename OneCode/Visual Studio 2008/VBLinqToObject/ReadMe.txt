/////////////////////////////////////////////////////////////////////////////
Use:

This example illustrates how to write Linq to Object queries using Visual 
Basic.NET. First, it builds a class named Person. Person inculdes the ID, Name 
and Age properties. Then the example creates a list of Person which will be 
used as the datasource. In the example, you will see the basic Linq operations 
like select, update, orderby, max, average, etc.


/////////////////////////////////////////////////////////////////////////////
Code Logic:

1. Define the Person class that includes the ID, Name, and Age properties.

2. Build a list of Person to be used as the data source.

3. Use Linq to perform the query operation.

4. Use Linq to perform the Update operation.

5. Use Linq to perform the Order operation.

5. Use Linq to perform the Max, Min, Average queries.

6. Use Linq to count the Person whose age is larger than 20.
